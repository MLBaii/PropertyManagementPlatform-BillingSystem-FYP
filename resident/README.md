# Module B — Resident Billing & Account Portal

The resident-facing mobile app of the PropertyBill system. Residents view itemised
bills, track account balances, download PDF bills/receipts, upload payment proof,
receive push notifications, and submit billing disputes.

**Stack:** React Native, Expo (managed workflow), TypeScript
**Backend:** consumes the shared ASP.NET Core Web API in `../backend/`
**Auth:** JWT stored via Expo SecureStore

## Feature modules
AuthModule · ProfileModule · BillsModule · AccountModule · PaymentProofModule · NotificationsModule · DisputeModule
